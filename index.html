明·言|运营商的霸王条款，就该让消费者买单？

运营商的霸王条款，就该让消费者买单？

重点：1、不允许自主选择使用所有资费方案；
　　　2、涉嫌欺诈消费者。

涉及公司：中国电信云南分公司

依据：

1、信息产业部关于保障移动电话用户资费方案选择权的通知 (信部清[2006]630号)： 二、在同一移动电话归属地内（俗称“移动电话本地网”），移动通信企业应保证本企业同一网络的原有用户，可以在不改变号码的情况下，自主选择使用本企业的所有资费方案（已停止发展用户的除外）。

2、中华人民共和国电信条例 第三十一条： 电信用户有权自主选择使用依法开办的各类电信业务。

　　先说说背景，作为一个四五年的电信“老用户”，一直都感觉电信的服务、资费、网速等甚合心意，也一直在推荐身边朋友来使用电信的服务。前一阵也曾想着换个价格略高的套餐，但因为近三个月中又一次因流量超出套餐导致平均消费不满足所谓的变更标准，后不了了之。直到前几天，因为个人的一些其他情况，电信的这张手机卡不再打算当做我的主力用卡，但用了四年的号码，说要注销难免也有些不舍，就想着再次尝试将资费转成月租较低的套餐，矛盾便从这时候开始了。

踢球技术哪家强，中国电信数第一。

　　上个月月底时候，我致电10000号客服热线，再次咨询转套餐一事，收到客服同样的回答，转套餐可以，需要根据近三个月的平均消费，只能高不能低。后来上网搜索相关资料发现，国家有明文规定不能限制用户转套餐。随后又打去第二个电话，摆明态度，客服回复让我去当地自营营业厅办理。
　　国庆假期，10月2日我前往昆明市离我最近的营业厅工人新村营业厅办理准备办理套餐变更业务。刚到营业厅，就受到工作人员热情的招呼，询问我要办什么业务。当我说需要变更套餐时，拿出一堆宣传材料并强调变更后的套餐月租只能比近三个月平均消费高。我能怎么办，我也很无奈。于是我再次说明我已经查过相关法律法规，我拥有选择套餐的自由。在办理业务中，工作人员表示，他们只能受理19元的套餐，并没有我所说9元的那个套餐。我当即用手机打开中国电信官网打开相关套餐页面拿给他看。工作人员竟告诉我，那你直接在网上办啊，跑这里来干什么？我？？？？？我特么能办我还打人工客服干嘛？能办我还来营业厅干嘛？这波球踢得我服！这要是进了国家队世界杯早夺冠了。

自相矛盾频食言，啪啪打脸我不光看着还得说话。

　　随后，当场拨打了云南省电信投诉电话，表明我想转套餐但是营业厅各种推脱不能受理。果然是投诉部门，再三交涉后，让我把电话拿给了营业厅工作人员，告诉了工作人员套餐代号并指导如何变更。可能由于权限问题，营业厅工作人员表示并搜不到相关套餐，于是，告诉我问题已记录，等处理吧，3-7个工作日。
　　不得不说电信很敬业，国庆假期也依然在处理我的投诉，只是画风好像越来越迷了，一开始打来电话说处理中了，让我耐心等待。后来打电话告诉我没有这个套餐了，让我办理营业厅所说的套餐。真的没有了吗？一个金牛座必然是要钻牛角尖的，当场我就开电脑逛官网、掏出手机上掌厅截图留念。还有什么说的？我告诉客服他们权限不够就向上一级汇报，他给我留个电话还让我自己打。打就打，打过去电话，依旧告诉我停办，让我截图发到QQ客服。又踢皮球么？附上官网和掌厅截图：

明·想|只想从个人的经历，谈谈创业

发表于 2017-08-18 | 分类于 明想 ， 创业 ， 随笔 | 评论数：

　　前几日，新闻播报了《习近平总书记给第三届中国“互联网＋”大学生创新创业大赛“青年红色筑梦之旅”的大学生的回信》，回信中“在创业中增长智慧才干，在艰苦奋斗中锤炼意志品质，在亿万人民为实现中国梦而进行的伟大奋斗中实现人生价值”一句，深深鼓舞着广大奋斗在创新创业一线的学生们。

　　我的母校云南大学滇池学院学生创业团队彩云本草作为云南省选拔的队伍，就是习总书记回信对象的一员，作为一名滇院人，我的内心不免感慨万千。

创业就是要猛志常在

　　说起来我也算是个老创业者了，从大二2014年后半年开始创业，到2015年首届“互联网+”大学生创新创业大赛和我的团队“采薇民绣”拿到国赛金奖，再到后来和朋友联合创办了云南坚果文化传播有限公司，“唯有牺牲多壮志，敢叫日月换新天”，做为一个创业者，我认为我们需要的就是毛主席诗中的这种豪心与壮志。

　　初次创业也是初次涉足文化行业，半途夭折，合伙的几个人赚了点小钱，但这种小商贩式的创业，因为没有完善的现代化企业管理制度，健全的财会制度，没法把事情持续性的做下去。合伙的几人，后面也就商量着就且这么着吧，边走边瞧。

　　2015年，正巧国家有了“大众创业，万众创新”的号召，我们又重新燃气了创业的梦想。从哪里跌倒就从哪里爬起，这次我们要涉足的方向依然是文化方向，我们赶着首届“互联网+”大学生创新创业大赛的浪潮，精心打磨了“采薇民绣”项目，一路过关斩将先后获得省赛冠军和国赛金奖。那时候，我大三，因为学业问题和业务方向的原因，留在了母校在学习创业园重新成立了云南坚果文化传播有限公司，向着我们的初心，继续做着切合我们自身兴趣的影视、广告业务。苦归苦，累归累，但这是我们的兴趣所在。业务平平撑过一年，我们几个合伙人内心也很是开心，那晚我记得是在学校外面的烧烤摊，我们吃着烧烤喝着啤酒，畅想着以后。

创业从没有一帆风顺

　　正如习总书记在回信中所说，“艰难困苦，玉汝于成。”想创业就得先吃苦，我和我的合伙人们，嘴里经常挂着这么一句话“创业没有假期，创业者没有个人时间。”工作起来，经常是忘了吃饭，忘了时间。在我们学校的创业园中，大晚上工作结束一出大楼，感觉已经挺晚了，但是回过头看，创业园的大楼依然灯火通明，大部分的创业团队，都在办公室加着班。他们和我们一样，也只是万万千千创业者之中的一员，他们都在那么拼着，我们也这么拼着，也只是挣扎在收支平衡的线上，那么广大的创业者们我什么不吃苦，不奋斗呢。也许我们这样奋斗，也不一定能成功，但是不奋斗，就根本没有办法成功。

　　当然，只有吃苦是远远不够的，还得屡败屡战、越挫越勇。我们学校这次冲击第三届“互联网+”大学生创新创业国赛的团队就是如此。如果彩云本草没有走出第二届大赛失利的阴影，没有从上一次的失败中总结经验砥砺前行，没有将他们的项目继续做下去，自然也无法在乌蒙山片区带动农户创就业，没有机会走出云南走进陕西，更不会因为参加“青年红色筑梦之旅”收到习总书记的回信。还有红糖馒头的团队，创始人罗三长从初中时候就开始自力更生，不忘学习，不断进取，也就没有如今的红糖馒头，也不会谱写下这么一段创业“罗馒史”。

　　不光是我们，任何一个“巨无霸”都是从失败中走出来的。阿里巴巴、京东、360，无不例外都是从努力奋斗和不断总结失败中来的。我一直很推崇马云那句“把错误变成营养”，而我从我们“滇院人”的身上，也看到了这种坚持，不放弃的精神吗，这也是为什么我们滇院人能不断进步的原因。

创业理应当及锋而试

　　如今创新创业的潮流下，这是最好的时代，也是最坏的时代。数据显示在去年，每天有1.5万家新公司诞生，3700家公司退出。“时来易失,赴机在速”。不得不说，现如今国家给出的政策是相当优惠，各大高校也不断推进创新创业教育改革。越来越多的大学生走进了创业的领域，机会与风险共存。如何能够抓住机遇，避开风险是每一家初创企业都所要面对的问题。

　　当然机会也不是你说抓就抓得住的。俗话说“打铁还需自身硬”，如过自身能力不够强，机会也会从眼前溜走的。正是如此，高校的创新创业教育尤为显得重要。当然也不是说机会就不重要，一方面，大学生储备好足够的创新创业知识，才不至于眼睁睁看着机会从手中溜走；另一方面，只有一次次的抓住机会，才会一步一步从渺小中成长。

　　改用影片《让子弹飞》中的一句台词，机会和能力对初创企业都不重要,抓住机会对初创企业很重要。

创业心自当波澜不惊

　　创业从来都不是一蹴而就的，所以在创业路上，一个良好的心态也是至关重要。就好比股市中的王牌操盘手，不管涨跌，都能保持一颗平常心来对待。就现在的我们的90后创业者而言，年轻既是资本也是短板。有想法、身体壮、能吃苦，但是往往因为年轻气盛、缺乏经验很容易给一个刚有起色的公司带来致命的打击。

　　在我看来，心态一定要正，创业几年，也整理了一点心得。

  • 赚钱虽然重要，但是以赚钱为目标是万不可取的
  • 目光长远，不能一味地只去追求眼前利益
  • 不能眼高手低，仰望星空也得脚踏实地。
  • 只说不做，也就只能是说说

　　最后一句话，与君共勉。但行好事，莫问前程；不忘初心，方得始终。

Markdown排版在微信公众号中的应用

发表于 2017-08-15 | 更新于 2019-04-19 | 分类于 Hexo笔记 ， Markdown ， 微信公众号 | 评论数：

昨天想起来似乎还有个闲置的公众号，当初是为了体验小程序开发而注册的。既然现在又写回了博客，就想着那顺便把文章一并也就发在公众号里了。

在把文章复制到微信公众号的时候发现了一个问题，那就是原声的微信公众号编辑器似乎不支持Markdown语法的渲染。作为一个懒人自然是不愿意去重新找个微信推文编辑器再做一次排版，况且个人而言Markdown的排版用着很顺手。

于是乎，求助于万能的google，google检索出来的信息，我筛选出了两种可以将Markdown语法直接发布在微信推文编辑中的方法。

两种方法

  • 使用Chrome浏览器插件Markdown Here
  • 使用MarkEditor直接导出为微信公众号文章格式

Markdown Here使用方法

Markdown Here算是一种非常无脑的解决方案了，非常简单。

你要做的只是安全上网去Chrome商店搜索安装Markdown here插件，然后将Markdown编辑的文本（带格式）复制到公众号文章编辑框里，按下快捷键Ctrl+Alt+M，你的Markdown就会被渲染出来。微调一下保存就ok了。

MarkEditor使用方法

如果选择MarkEditor的话，那么恭喜你，你可能Get到了最简单的方法。

MarkEditor本身就是一款Markdown文本编辑器，它的强大之处在于可以直接导出微信公众号文章格式。当你在MarkEditor里面以标准的Markdown语法编辑好文档后，直接在软件导出功能中就可以导出了。导出之后，复制在公众号文章编辑框里微调保存即可。

  • 附MarkEditor地址：MarkEditor官网

                宅在办公室的五一假期

　　五月二日，假期的第二天，20岁的第二天。雨。
　　今年春城，早些时候的大雪，以及不久前的高温给了我太多的意外。而今天的雨水，还回了春城本该有的温度。   


　　这个假期，我没有选择旅行。没有原因，只是一种说不出的感觉。睡到自然醒，揉揉惺忪的睡眼。不觉已至正午。翻身，起床。呆呆地看着窗外，心中突然冒出洗个冷水澡的念头。冰冷的水流在身上滑落，虽说不上有多么刺骨，但也难免带来一丝丝寒意。


　　洗漱完毕，无处可去的我选择了办公室。我喜欢这里，一个自由、安静的角落。能够安心思考，静心阅读。坐在窗边，打开豆瓣FM，任凭窗外雨水滴答，聆听忧伤的民谣。不觉思绪回到了高中时代那最后的岁月。距离2014年高考仅剩36天了，忆起去年的这个时候，那痛并快乐着的青春，并不似如今般无所事事，而如今空虚散漫的生活再也没有的往日的充实和满足。


　　去年的背井离乡，义无反顾地踏上了南下的火车。不觉来到云南这片土地已经近十月之余。失落过，无助过，伤心过，迷茫过，也曾收获过。永远忘不了去年七月的艰难抉择。理想败给现实，选择了不怎么喜欢的会计专业。可是现在已经无力去改变，之有继续走下去。


　　也许，这就是人生，你永远不能按照你的喜好来做选择。也许这就是青春，过去的，已成定局。




搬运于2014年05月02日QQ空间博文。


          
        
      
    


    

    
    
    

    

    
      
    
    

    

    

      

      

      

      
      
        

      
    

  

  
  
  
  



    
      

  

  
  
  

  

  

  
  
  
  

    

    
      
      
      
    

    
      
    

    
      


        
        
          

                
                
                年復一年

　　虽说对舶来的西洋节日毫无兴趣，可那又如何？平安夜平安度过，圣诞节平平淡淡，转眼已是岁末，2015走了，2016来了，该来的总归会来。不觉匆匆又一年，时间都去哪了？




“燕子去了，有再来的时候；杨柳枯了，有再青的时候；桃花谢了，有再开的时候。”——朱自清·《匆匆》  




　　可青春呢？时光呢？不觉二十馀载的人生已经过去了，一生，又有多少个二十馀载？


　　回望这年，得与失，成与败，似乎并没有那麽重要了。经历太多，得到太多，失去太多。到头来得非所想，失非所愿，只不过痛苦的执念罢了。不堪回首的一年，不，应该是不堪回首的岁岁年年。


　　昆明最近的天气不是很好，心情也似天气一样糟糕。看著朋友圈中2015的年终总结，2016的新年愿望，只是微微一笑，毕竟有梦总是好的。大抵是一年的疲惫在这一刻集中的爆发，什麽事都不想想，什麽事都不乐意做，只想安安静静的，发发呆，喝喝小酒，抽抽闷烟。往事如烟，一部分吸进肺裡默默承受，一部分，随著空气，烟消云散。


　　得过且过了，2016，坐等考完最后两科，收拾心情，回家过年。虽然回忆依旧是那麽不堪，可哥们儿的雄心，哥们儿的壮志还有一丝丝存在心中。大道无为，2016顺其自然吧。“随分良聚，堪对此景，争忍独醒归去。”




搬运于2015年12月21日QQ空间博文。
